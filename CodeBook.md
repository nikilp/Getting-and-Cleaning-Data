# Code Book

This document describes the summarised variables in the tidy data set `summarised_data.txt`.

### Identifiers
* `1` | `subject` - Subject ID; this takes values between 1 and 30 - the number of subject in taking part in data collection.
* `2` | `activity` - Activity label; this describes the type of activity performed by the subject. It corresponds to:

| Activity           | Value  |
| ------------------ |:---:|
| walking            |  1  |
| walkingUpstairs    |  2  |
| walkingDown        |  3  |
| sitting            |  4  |
| standing           |  5  |
| laying             |  6  |
